Privacy Policy

Effective date: April 26, 2026

CoworkX is a macOS productivity app that runs in the macOS App Sandbox. The app is designed to store local workspace data in the app container unless you choose a file location through macOS file access controls. Your privacy is important — this policy explains what information CoworkX accesses and how it is handled.

Information You Provide

CoworkX may store API keys, tokens, workspace entries, snippets, calendar-related configuration, and local database content that you enter into the app. API keys and tokens are stored in the macOS Keychain.

Data Storage

Local app data is stored on your Mac. CoworkX does not sell personal data. CoworkX does not include third-party advertising tracking.

Network Access

CoworkX uses outgoing network connections only for user-initiated or user-configured workflows, such as:

  • Connecting to a local AI endpoint (e.g., Ollama)
  • Fetching RSS feeds
  • YouTube API workflows
  • Analytics integrations
  • Subscription purchase validation through Apple

No network connections are made without your knowledge or configuration. The app does not phone home, send analytics, or transmit personal data to any server not explicitly configured by you.

Calendar Access

CoworkX requests calendar access only when you use calendar features. Calendar access is used to query or create events that you request. CoworkX does not access your calendar data without your explicit action.

In-App Purchases

CoworkX uses Apple in-app purchases for auto-renewable subscriptions. Subscription purchase and billing information is handled entirely by Apple. CoworkX does not receive or store your payment information.

Data Security

CoworkX is built on Apple's security frameworks:

  • App Sandbox — The app operates within macOS security boundaries.
  • Keychain — API keys and tokens are stored in the encrypted macOS Keychain.
  • Local Storage — All app data is stored locally on your device.

Third-Party Services

CoworkX does not embed third-party analytics, advertising, or tracking SDKs. The app connects only to services you explicitly configure (e.g., your own YouTube API key, your own local AI endpoint, your own RSS feeds).

Changes to This Policy

If this privacy policy is updated, the effective date at the top of this page will be revised. Continued use of CoworkX after changes constitutes acceptance of the updated policy.
