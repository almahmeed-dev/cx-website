Built with privacy in mind

🔒 Sandboxed

The Mac App Store build runs in the macOS App Sandbox. Your data stays where it belongs — on your device.

🔑 Keychain-Backed

API keys and tokens are stored in the macOS Keychain, not in plaintext files. You control your credentials.

📁 Local by Default

All app data is stored in the app container unless you explicitly choose another location through macOS file controls.
